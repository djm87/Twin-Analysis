vec42quat

returns a quaternion q with q u_1 = v1 and q u2 = v2

Description

The method vec42quat defines a quaternion q by 4 directions u1, u2, v1 and v2 such that q * u1 = v1 and q * u2 = v2

Syntax

q = vec42quat(u1,v1,u2,v2)

Input

u1, u2

vector3d
v1, v2

vector3d

Output

q

quaternion
