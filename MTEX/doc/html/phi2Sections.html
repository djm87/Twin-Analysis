phi2Sections

phi2 sections for ODF and orientation plotting

Syntax 

oS = phi2Sections(cs1,cs2,'sections',5)
oS = phi2Sections(cs1,cs2,'phi2',(0:15:90)*degree)

Input

cs1, cs2

crystalSymmetry, specimenSymmetry
param,val Parameters and values that control phi2Sections
Parameter Description
'sections'

number of sections
'phi2'

explicite section values
