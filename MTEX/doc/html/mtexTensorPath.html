mtexTensorPath

returns the default path to tensor-files

Syntax

mtexTensorPath
getMTEXpref('TensorPath')
setMTEXpref('TensorPath',path)
getMTEXpref('TensorPath') long form of mtexTensorPath
setMTEXpref('TensorPath',path) change the path to Tensor files
