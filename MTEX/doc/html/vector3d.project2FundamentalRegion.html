project2FundamentalRegion

(method of vector3d)

projects vectors to the fundamental sector of the inverse pole figure

Syntax 

v = project2FundamentalRegion(v,cs)
v = project2FundamentalRegion(v,cs,'antipodal')
v = project2FundamentalRegion(v,cs,center)

Input

v

vector3d
cs

symmetry
center

vector3d
param,val Parameters and values that control project2FundamentalRegion
Parameter Description
'antipodal'

include antipodal symmetry

Output

v

vector3d
