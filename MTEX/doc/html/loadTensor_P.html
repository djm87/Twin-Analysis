loadTensor_P

load a Tensor from a file

Description

loadEBSD_generic is a generic function that reads any ascii file containing a matrix like 

e_11 e_12  ... e_1j
 .     .   ...  .
 .     .    .   .
e_i1   .   ... e_ij

describing the a Tensor

Syntax 

T   = loadTensor_generic(fname,)

Input

fname

file name (text files only)
param,val Parameters and values that control loadTensor_P
Parameter Description
'name'

name of the tensor
