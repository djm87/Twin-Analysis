ChristoffelTensor

(method of stiffnessTensor)

Christoffel tensor of an elasticity tensor for a given direction

Formula: E_jk = C_ijkl n_j n_l

Syntax

E = ChristoffelTensor(C,n)

Input

C

elatic stiffness tensor
x

list of vector3d

Output

E

Christoffel tensor
