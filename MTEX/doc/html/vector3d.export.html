MTEX - A MATLAB Toolbox for Quantitative Texture Analysis View Code

export

(method of vector3d)

export quaternions to a ascii file

Syntax

export(v,fname,'param1',val1,'param2',val2)

Input

v

vector3d
fname

filename
param,val Parameters and values that control export
Parameter Description
'polar'

export polar coordinates
'xyz'

export x, y, z coordinates (default)
'degree'

output in degree (default)
'radians'

output in radians
