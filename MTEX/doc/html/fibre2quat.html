fibre2quat

arbitrary quaternion q with q * h = r

Description

The method fibre2quat defines a list of quaternions q by a crystal direction h and a specimen direction r such that q * h = r

Syntax

q = fibre2quat(h,r,'param1',val1,'param2',val2)

Input

h

Miller or vector3d
r

vector3d
param,val Parameters and values that control fibre2quat
Parameter Description
'resolution'

double

Output

q

quaternion
